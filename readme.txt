Customize your WordPress Login Page. Upload new logo, change easily background color, and much more!

== Description ==
Design your website login page. Upload your logo, change background color, add custom messages for your users.

My WP Login allow you to customize 100% your website login page as you would like.

It also provides a custom css section for designers to add custom css for login page.

Features include:

*   Upload custom logo to login page.
*   Specify you logo width and height.
*   Add a custom logo url and custom title attribute for logo.
*   Upload a background image for login page.
*   Select background color for login page.
*   Customize login form by changing background color, fields color, labels color, button text color and button text color.
*   Split login page in two vertical views. If this option is checked the background image will go to the left.
*   Add custom message in login page.
*   Add your own css to the WordPress Login Page.

== Installation ==
= Installing manually: =

1. Upload the plugin files to the `/wp-content/plugins/mywplogin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go to *MyWPLogin > Home* in the left-hand menu to configure the plugin
